# 🚀 Quick Start Guide

## Get Started in 3 Steps

### 1. Install Dependencies
```bash
cd asif-portfolio
npm install
```

### 2. Start Development Server
```bash
npm start
```

Your portfolio will open at **http://localhost:3000**

### 3. Build for Production
```bash
npm run build
```

## ⚡ Quick Customization

### Update Your Information

1. **Personal Info** - Edit `src/components/Hero.js`
   - Name, title, description
   - Social media links

2. **About Section** - Edit `src/components/About.js`
   - Professional summary
   - Statistics

3. **Skills** - Edit `src/components/Skills.js`
   - Add/remove skills
   - Update skill levels

4. **Projects** - Edit `src/components/Projects.js`
   - Add your projects
   - Update images, links, descriptions

5. **Experience** - Edit `src/components/Experience.js`
   - Work history
   - Education

6. **Contact** - Edit `src/components/Contact.js`
   - Email, phone, location
   - Social links

### Change Colors

Edit `tailwind.config.js`:
```javascript
colors: {
  primary: '#3b82f6',    // Change to your color
  secondary: '#8b5cf6',  // Change to your color
}
```

### Add Your CV

Place your CV PDF in `public/` folder as `Asif-Nazeer-CV.pdf`
